### About me:
Hi, I'm Milene, but you can call me just Mi.

I'm a Software Engineer - Frontend Developer from Brazil. 

I have been working with frontend development since 2018. Since I started I'm always looking for ways to improve my skills and I already had worked in some companies with different technologies and tools. I like to learn new things and for me the best way to learn something is putting it into practice.

I've been wanting to contribute to open source projects, so lately I've been looking for some ways to do that.

Besides, I love music and I'm studying english so I can talk with many other people from the rest of the world! =)
